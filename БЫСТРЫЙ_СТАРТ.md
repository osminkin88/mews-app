# ⚡ Mews — Быстрый старт

## 1. Установка

- Открой файл **Mews-1.0.0-universal.dmg**
- Перетащи **Mews** в **Applications**

## 2. Если macOS блокирует

Кликни правой кнопкой на Mews.app → **«Открыть»** → **«Открыть»**

Или: **Системные настройки** → **Конфиденциальность и безопасность** → **«Всё равно открыть»**

## 3. Подключение

- При запуске нажми **«Запустить Chrome»**
- Войди в аккаунт Higgsfield в Chrome
- Вернись в Mews — статус станет зелёным ✅

> Нужен установленный Google Chrome.

## 4. Генерация

1. Создай проект на экране **Проекты**
2. На экране **Настройки** нажми **«Скачайте шаблон»** и заполни промпты
3. Загрузи файл через зону **«Загрузить промпты»**
4. Выбери модель, качество, формат
5. Нажми **«Запустить генерацию»** ▶

## 5. Результат

Изображения сохраняются в:
```
~/Documents/Mews/название-проекта/
```
