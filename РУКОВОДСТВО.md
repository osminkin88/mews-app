# 🐱 Mews — Руководство пользователя

> AI-генератор изображений. Десктопное приложение для macOS.

---

## 📦 Установка

### Из DMG-файла

1. Дважды кликни на `Mews-1.0.0-universal.dmg`
2. В открывшемся окне перетащи **Mews.app** в папку **Applications**
3. Закрой окно установки
4. Открой **Mews** из папки Приложения (Applications) или через Launchpad

### ⚠️ «Приложение от неизвестного разработчика»

macOS может предупредить о том, что приложение от неустановленного разработчика. Это нормально — у приложения пока нет сертификата Apple.

**Как открыть:**

1. Открой **Системные настройки** → **Конфиденциальность и безопасность**
2. Пролистай вниз — увидишь сообщение «Mews заблокирован»
3. Нажми **«Всё равно открыть»**
4. Подтверди в диалоге

Или быстрый способ:
- Кликни правой кнопкой на Mews.app → **«Открыть»** → **«Открыть»** в диалоге

После первого запуска macOS больше не будет спрашивать.

## 🚀 Первый запуск

### 1. Подключение к Higgsfield

При первом запуске приложение покажет экран **Подключения**.

- Нажми **«Запустить Chrome»** — откроется Google Chrome с сайтом Higgsfield
- Войди в свой аккаунт на Higgsfield (через Google или Email)
- Вернись в Mews — подключение определится автоматически
- Зелёный статус ✅ = всё работает

> **Требование:** На Mac должен быть установлен Google Chrome.

### 2. Создание проекта

- Перейди на экран **Проекты** (иконка в боковом меню)
- Нажми **«+ Новый проект»**
- Введи название (например: «Тест баннеров»)
- Проект создан — он появится в списке

## 📋 Основной рабочий процесс

### Шаг 1: Подготовь промпты

Mews генерирует изображения по текстовым промптам. Промпты загружаются из Excel-файла.

1. На экране **Настройки** нажми ссылку **«Скачайте шаблон»**
2. Выбери куда сохранить — скачается готовый Excel-файл с примерами
3. Открой файл и заполни свои промпты
4. Сохрани файл

**Столбцы шаблона:**

| Столбец | Обязательный? | Описание |
|---------|:---:|----------|
| **ID** | да | Номер промпта (001, 002, ...) |
| **Промпт** | да | Текст промпта на английском |
| **Комментарий** | нет | Любая заметка для себя |

**Формат файла:** `.xlsx` или `.csv`

### Шаг 2: Импортируй промпты

1. Перейди на экран **Настройки**
2. Кликни на зону **«Загрузить промпты»** и выбери свой Excel/CSV файл
3. Промпты загрузятся как набор (сет)
4. Можно добавлять несколько файлов — кнопка **«Добавить новый»** появится после первого импорта
5. Переключаться между наборами можно через чипы с названиями

### Шаг 3: Настрой параметры

На том же экране **Настройки** (заголовок «Подготовка к запуску»):

| Параметр | Что это |
|----------|---------|
| **Модель** | AI-модель для генерации (∞ Unlimited) |
| **Качество** | Разрешение: 2K, 4K и т.д. |
| **Формат** | Пропорции изображения: 1:1, 16:9, 9:16 и т.д. |
| **Вариантов на промпт** | Сколько изображений на каждый промпт (1, 2 или 4) |

### Шаг 4: Запусти генерацию

1. Внизу экрана Настройки показана сводка: сколько промптов × вариантов = итого
2. Нажми кнопку **«Запустить генерацию»** ▶
3. Приложение автоматически перейдёт на экран **Генерация**
4. Прогресс отображается в реальном времени — видно каждый промпт и статус
5. Можно остановить в любой момент кнопкой **«Стоп»**

> Генерация одного изображения занимает ~30–90 секунд.

### Шаг 5: Отбери лучшие

1. Перейди на экран **Отбор** (🖼)
2. Для каждого промпта покажутся сгенерированные варианты
3. Кликай на понравившиеся — они пометятся ✅
4. Отобранные сохранятся в отдельную папку

### Шаг 6: Посмотри результаты

- Экран **Результат** покажет общую статистику по проекту
- Там же можно открыть папку с файлами на диске

## 📁 Где хранятся файлы

| Что | Где |
|-----|-----|
| **Сгенерированные изображения** | `~/Documents/Mews/` (папка «Документы») |
| **Каждый проект** | `~/Documents/Mews/название-проекта/` |
| **Сгенерированные** | `.../sets/prompt-set-001/generated/` |
| **Отобранные** | `.../sets/prompt-set-001/selected/` |
| **Настройки приложения** | `~/Library/Application Support/mews/` |
| **Сессия Chrome** | `~/Library/Application Support/mews/session.json` |

> Папку с изображениями можно изменить в настройках.

## 🔧 Решение проблем

### Приложение не открывается (macOS блокирует)

→ Смотри раздел «Приложение от неизвестного разработчика» выше.

### Потеря подключения к Chrome

1. Проверь, что Chrome запущен
2. В приложении нажми **«Переподключить»** на экране Подключения
3. Если не помогает — нажми **«Запустить Chrome»** заново

### Промпты не импортируются

- Файл должен быть `.xlsx` или `.csv`
- В файле обязательно должен быть столбец **Промпт** (или `prompt`)
- Кодировка CSV: UTF-8
- Если Excel — сохрани как `.xlsx` (не `.xls`)
- Скачай и используй шаблон из приложения — он гарантированно работает

### Генерация зависла

- Нажми **«Стоп»**
- Подожди несколько секунд
- Попробуй снова — приложение продолжит с последнего промпта

### Chrome не запускается

- Убедись, что Google Chrome установлен в `/Applications/`
- Попробуй закрыть все окна Chrome и запустить заново из Mews
